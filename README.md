This is the repository for my end of 1st year research project for university.
If you came across this randomly it's code for a permutations test to examine the impact of leitmotif on the success of a broadway musical.
I wasn't able to find a particularly good dataset so the result could be caused by bias accross the whole population of musicals not just the top 40 that I tested.
Put all the csvs in a folder named data on the same file level as the code and it should work
